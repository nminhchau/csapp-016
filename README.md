## Book structure

### Part I: Program Structure and Execution

- Chapter 1 introduces the system stack through the lifecycle of `hello.c`.
- Chapter 2 covers bit-level representations, integer arithmetic, and floating point.
- Chapter 3 explains x86-64 machine code, data movement, control flow, procedures, arrays, structures, and floating-point code.
- Chapter 4 builds a simplified Y86-64 processor and introduces pipelining.
- Chapter 5 focuses on code optimization and performance bottlenecks.
- Chapter 6 explains storage technologies, locality, cache memories, and cache-friendly programming.

### Part II: Running Programs on a System

- Chapter 7 covers static linking, dynamic linking, relocation, symbol resolution, object files, and library interposition.
- Chapter 8 explains exceptions, processes, system calls, signals, nonlocal jumps, and process-control tools.
- Chapter 9 covers virtual memory, address translation, memory mapping, dynamic allocation, garbage collection basics, and common memory bugs.

### Part III: Interaction and Communication between Programs

- Chapter 10 explains Unix I/O, files, metadata, directories, sharing, redirection, and standard I/O.
- Chapter 11 covers client-server programming, IP, DNS, sockets, web servers, and the Tiny Web Server.
- Chapter 12 covers process-based, event-driven, and thread-based concurrency, shared variables, semaphores, thread safety, races, and deadlocks.
